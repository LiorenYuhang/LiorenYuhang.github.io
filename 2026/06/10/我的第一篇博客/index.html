你好，世界！

这是我的第一篇博客文章 🎉

今天我开始用 Hexo + Butterfly 搭建自己的个人网站，托管在 GitHub Pages 上。

为什么要建个人网站？

  • 📝 记录学习和成长
  • 🚀 展示项目和作品
  • 🌐 拥有自己的互联网角落

搭建过程

  1. 安装 Node.js 和 Hexo
  2. 选择 Butterfly 主题
  3. 写第一篇文章（就是这篇！）
  4. 部署到 GitHub Pages

这只是开始，后面会慢慢完善网站内容和样式。
